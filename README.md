# 🏦 CreditWise: Loan Approval Prediction using Machine Learning

## 📌 Overview

CreditWise is an end-to-end supervised Machine Learning project designed to predict whether a loan application will be approved based on an applicant's financial, demographic, and credit-related information.

The project follows a complete machine learning workflow, including data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and performance evaluation. Multiple classification algorithms were implemented and compared to identify the most effective model for loan approval prediction.

---

## 🚀 Key Highlights

* Built a complete supervised Machine Learning pipeline for binary classification.
* Performed extensive Exploratory Data Analysis (EDA) to uncover patterns and relationships within the data.
* Handled missing values, encoded categorical variables, and applied feature scaling techniques.
* Engineered additional features to improve model performance and capture non-linear relationships.
* Trained and compared multiple classification algorithms:

  * Logistic Regression
  * K-Nearest Neighbors (KNN)
  * Gaussian Naive Bayes
* Evaluated models using industry-standard metrics:

  * Accuracy
  * Precision
  * Recall
  * F1 Score
  * Confusion Matrix
* Selected the best-performing model based on classification performance and business relevance.

---

## 🎯 Problem Statement

Financial institutions receive thousands of loan applications every day. Manual evaluation can be time-consuming and prone to inconsistencies.

The objective of this project is to develop a machine learning solution capable of automatically predicting loan approval decisions using applicant information, helping streamline the lending process and support data-driven decision making.

## 🤖 Models Implemented

### Logistic Regression

A linear classification algorithm used as a strong baseline model for binary prediction tasks.

### K-Nearest Neighbors (KNN)

A distance-based algorithm that classifies applicants based on similarity to existing data points.

### Gaussian Naive Bayes

A probabilistic classification model based on Bayes' Theorem and feature independence assumptions.
